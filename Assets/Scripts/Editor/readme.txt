How To Use Tim's Amazing Wall Generator

- Create a GO called "Floor" with a Z value of 1, and a scale of (1, 1). Edit it as you like.
- Create a GO called "Wall" with a Z value of 0, and a scale of (1, 1). Edit it as you like.

- Open the wall generator window GameObject > Tim's Amazing Wall Generator
	- Select the csv file you want to read from. Press GENERATE
	- Make sure the CSV is not open in Excel at this time or Windows will not allow you to read it
- The walls will be generated under "Wall Daddy" GO. The floor will be generated separately.
- Remove the Floor and Wall objects if you are satisfied.
